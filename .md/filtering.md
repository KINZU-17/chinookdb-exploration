# Data Filtering in SQLite
- Filtering allows you to select specific rows based on conditions using the WHERE clause.

## Key Concepts

- **`WHERE` Clause**: Filters rows based on specified conditions
- **Comparison Operators**: Supports `=`, `!=`, `<`, `>`, `<=`, `>=`, `LIKE`, `IN`, and `BETWEEN`.
